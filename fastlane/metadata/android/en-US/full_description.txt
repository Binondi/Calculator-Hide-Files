<p><b>Calculator Hide File App</b> is an open-source Android application that combines a functional calculator interface with a secure private file storage feature. It allows users to store and manage personal files such as images, videos, documents, and notes, protected by a passcode.</p><p><b>Key Features:</b></p>
<ul>
<li><b>Dual Functionality:</b> A fully working calculator alongside a private file storage area accessible via passcode.</li>
<li><b>Passcode Protection:</b> Access your private files securely using a user-defined code.</li>
<li><b>Secure File Manager:</b> Add, remove, and organize files within a protected local environment.</li>
<li><b>Local Storage:</b> All files are stored on-device, ensuring user control and privacy.</li>
<li><b>Lightweight & Fast:</b> Designed to run smoothly across a wide range of Android devices.</li>
<li><b>No Root Required:</b> Works on standard Android devices without requiring root access.</li>
<li><b>File Encryption:</b> Protect stored data using encryption mechanisms.</li>
</ul><p><b>How It Works:</b></p>
<ol>
<li><b>Calculator Mode:</b> Use the app as a standard calculator for everyday calculations.</li>
<li><b>Access Private Storage:</b> Enter your passcode to open the secure file storage area.</li>
<li><b>Manage Files:</b> Import, organize, and remove files within the app. Data remains protected locally on your device.</li>
</ol><p><b>Requirements:</b></p>
<ul>
<li>Android 6.0 or higher</li>
<li>Storage permissions enabled</li>
<li>“Manage All Files” permission may be required on Android 11+ for full file access functionality</li>
</ul><p><b>Technologies Used:</b> Kotlin, XML, Android SDK, MediaStore API, and secure local storage techniques</p><p><b>Open Source:</b> This project is fully open-source, allowing users to inspect the code and verify how data is handled.</p><p><b>Privacy Focus:</b> The app operates entirely offline and does not collect or share user data.</p>
